---
title: plus
---

`plus` adds the number passed in to the input number.

```liquid
{% raw %}
{{ 4 | plus: 2 }}
{% endraw %}
```

```text
{{ 4 | plus: 2 }}
```

```liquid
{% raw %}
{{ 16 | plus: 4 }}
{% endraw %}
```

```text
{{ 16 | plus: 4 }}
```

```liquid
{% raw %}
{{ 183.357 | plus: 12 }}
{% endraw %}
```

```text
{{ 183.357 | plus: 12 }}
```
