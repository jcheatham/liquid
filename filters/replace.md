---
title: replace
---

`replace` replaces every occurrence of the first passed-in string in the input string with the second passed-in string.

```liquid
{% raw %}
{{ "Take my protein pills and put my helmet on" | replace: "my", "your" }}
{% endraw %}
```

```text
{{ "Take my protein pills and put my helmet on" | replace: "my", "your" }}
```
