---
title: modulo
---

`modulo` returns the remainder when the input is divided by the passed-in parameter.

```liquid
{% raw %}
{{ 3 | modulo: 2 }}
{% endraw %}
```

```text
{{ 3 | modulo: 2 }}
```

```liquid
{% raw %}
{{ 24 | modulo: 7 }}
{% endraw %}
```

```text
{{ 24 | modulo: 7 }}
```

```liquid
{% raw %}
{{ 183.357 | modulo: 12 }}
{% endraw %}
```

```text
{{ 183.357 | modulo: 12 }}
```
